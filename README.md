🛠️ 현수의 작업실 (Studio)
현수의 작업실은 SKALA 과정에서 만들어가는 다양한 웹 프론트엔드 프로젝트와 실험들을 기록하는 개인 포트폴리오 겸 작업 공간입니다. 
HTML, CSS, JavaScript를 활용하여 기본적인 웹 페이지 구조를 설계하고, 레이아웃 시스템을 구축하며, 동적인 데이터 처리와 미니 게임을 직접 구현해 보는 것을 목표로 합니다.  

📌 주요 페이지 및 기능
🏠 메인 허브
각 하위 페이지로 이동할 수 있는 내비게이션을 제공합니다. 
HTML, CSS, JavaScript, Python 등 주차별 학습 기록과 향후 학습 계획을 기록합니다.  
실시간 날씨, 시계, 오늘의 뉴스 섹션 등 다양한 외부 데이터를 연동하는 기능이 포함되어 있습니다.  

👤 나의 프로필 
저의 간단한 자기소개와 좋아하는 음식, 취미 등을 리스트와 토글 형태로 제공합니다.  
올해 목표와 본인을 나타내는 키워드를 설명합니다.  

📅 나의 수업 
일주일 간의 전체 강의 스케줄을 가독성 높은 표 형태로 제공합니다.  

🏖️ 나의 휴일 
휴일의 시간대별 일과를 정리한 페이지입니다.  
체크박스를 클릭하면 취소선이 그어지는 애니메이션이 적용되어 진행도를 직관적으로 확인할 수 있습니다.  

✈️ 나의 여행 일기
오사카, 교토, 도쿄 등 잊지 못할 일본 여행의 사진과 맛집 정보를 카드 레이아웃으로 보여줍니다.  
배경 음악과 여행 브이로그 영상 등 멀티미디어 요소를 적극 활용했습니다.  

📝 회원가입
아이디, 비밀번호, 이메일, 생년월일, 관심 분야, 약관 동의 등 다양한 폼 요소를 활용한 회원가입 페이지입니다. 
폼 제출 후 이동하는 가입 환영 완료 페이지를 포함합니다.  

🎮 미니 게임 라운지메인 페이지 우측 사이드바를 통해 다양한 미니 게임과 유틸리티 기능을 제공합니다.  

🧱 벽돌 깨기 
HTML Canvas를 활용하여 직접 방향키로 패들을 조작하는 아케이드 게임입니다.  

⭕ 틱택토 ❌
메인 화면 내에서 곧바로 X와 O를 번갈아 가며 플레이할 수 있는 보드게임입니다.  

기타 유틸리티: 업다운 게임, 성적 계산기, 가방 속 물품 리스트 조회, 랜덤 명언 자판기 등의 인터랙티브한 요소를 포함하고 있습니다.  

🛠️ 사용 기술
시맨틱 태그(header, nav, main, aside, section 등)를 적극 사용하여 웹 표준을 준수합니다.  
CSS3: Flexbox와 Grid를 활용한 반응형/적응형 레이아웃, style.css를 통한 일관된 컬러 팔레트와 부드러운 Hover 애니메이션을 적용했습니다.  
JavaScript: DOM 조작, 이벤트 리스너, 객체와 배열을 활용해 미니 게임 로직과 UI 인터랙션을 구현했습니다.
